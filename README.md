This repository contains my solutions and practice exercises completed as part of the Cognizant Digital Nurture 5.0 (DN 5.0) Java Full Stack Engineer (Java FSE) Upskilling Program.

The objective of this upskilling program is to strengthen foundational and practical skills in modern web development and Java programming through hands-on exercises and self-paced learning.
